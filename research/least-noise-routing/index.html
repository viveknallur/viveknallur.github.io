Least Noise Routing

Screenshots of the SmartHopper in Action

SmartHopper fuses the routing engine, provided by the GraphHopper folks, with the participatory noise data collected by citizens.
Here’s what it looks like without the noise data being used:

Then, add the noise overlay created from the data uploaded from the smartphones of participating citizens

From the menu on the left hand side, change the weighting to Least Noisy

There, you have it. Noise has been added and used as a data source, for making routing decisions. Of course, the decision is only as good as the data it is based on.  So, the more data that is contributed by citizens, the better the routing decision can be.
